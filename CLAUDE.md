# Portfolio Website - Claude Code Configuration

## Project Overview
This is a single-page portfolio website for Samuel Devdas, showcasing AI & Data Science expertise. The site is built with pure HTML, CSS, and JavaScript.

## Development Commands
Since this is a static HTML website, no build tools or package managers are required.

### Local Development
- Open `website-samuel-portfolio.html` directly in a web browser
- Use a local web server for testing: `python -m http.server 8000`

### File Structure
- `website-samuel-portfolio.html` - Main portfolio page
- `stagewise-integration-code.html` - Additional code file
- `CV_Samuel_Devdas_G-8-10.md` - Resume in markdown format
- `extensions.json` - VS Code extensions configuration

## Key Features
- Responsive design with mobile optimization
- Dark/light theme toggle with system preference detection
- Smooth scroll animations and intersection observers
- 3D card hover effects
- Custom cursor with blend modes
- Video background in hero section
- Professional sections: About, Skills, Experience, Projects, Education, Contact

## Code Quality
- Pure HTML/CSS/JS (no frameworks or build tools)
- Semantic HTML structure
- CSS custom properties for theming
- Accessible design with skip links and ARIA labels
- Modern CSS features (Grid, Flexbox, backdrop-filter)

## Browser Compatibility
- Modern browsers supporting CSS Grid, backdrop-filter, and ES6
- Graceful fallbacks for older browsers
- Mobile-first responsive design
